What is HTTP and HTTPS?


HTTP and HTTPS are a protocol at the start of a URL

What is a URL?

URLs are a type of URI. A URI is a Uniform Reousrce Identifier which is a line of text
that identifies an online resource like a web page. A URL is used for identifying but
also retrieving online resources. Below an image of Facebook's URL with the HTTPS
protocol at the start.

What is a Protocol?

A protocol defines the communication mechanism to retrieve the resource such as a webpage.
The communication would consist of messages being sent back and forth between a server and
a client. The mechanism would be how the client and server would handle the messages.

What does HTTP and HTTPS stand for?

HTTP stands for "Hyper Text Transfer Protocol" and HTTPS stands for the same thing, but the
“S” stands for "Secure". The term “Hyper Text” is used to refer to a document that contains
hyperlinks to navigate around it, this is also what a website is.

HTTP vs HTTPS

HTTPS uses encryption and validation techniques when the server and client are communicating.
The most popular way it does this does this by obtaining a Transport Layer Security (TLS) certificate.
Click here for more information about TLS certificates.

HTTP does not use encryption when transmitting data, therefor it is visible to anyone monitoring
the connection. Almost all web pages use HTTPS, the ones that use HTTP are public webpages that
contain no sensitive information. If your using chrome, you will see a Not Secure statement before
the URL if the website is using HTTP. You should avoid entering any sensitive information if you
see that statement. An example is shown below.

The Client and Server

For our purposes a client is a web browser running on a device. This can be a windows laptop running
google chrome. A Server is the device from where the pages being browsed are stored. This could be
a windows laptop, but most likely is a desktop computer designed to store and process lots of data,
the computer may also be running Linux instead of windows. HTTP or HTTPS is the method of communication
between the client and the server.
