How is a web page downloaded?


To awnser this I will be using the following webpage: https://en.wikipedia.org/wiki/Morty_Smith

This webpage is downloaded over a connection between a client and the sever hosting the webpage,
in this case it would be Wikipedia's server. The content being downloaded would be hypertext
documents, images, and other files.

By using chromes tools, we can see a sequence of requests, and since we are using the HTTPS
protocol, they are HTTPS requests. Below is an image showing 18 requests being made between my
device (a windows computer) and Wikipedia's server.

If we select the Morty_Smith.png request we can see where the website gets the image from,
in this case the image is stored on Wikipedia's server. If we go to this URL it takes us to
the file Morty_Smith.png on Wikipedia's server, this will be presented as a blank webpage
that only shows an image of Morty.

The other requests we can see are images for the page logo and page symbols, there are also PHP
files which are used to generate HTML code, this is what webpages runs on.
