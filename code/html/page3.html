What is a request?


In the previous page we discussed how a webpage is downloaded, and that requests are made to
download the page contents on our device from the server. The things that can be requested
are hypertext documents, images, and other files. Now for the client to request these files,
the client will click on a link and the web browser will generate and send some text to the
server, this text would contain a get request method.

Two main types of request methods

GET request method. This is usually used to get a resource, these resources can be
hypertext documents, images, and so on, these are identified by the URI.

POST request method. This modifies an entity with existing URI on the server
according to the data in the response body. An example would be when posting
a comment to an article.

  • Successful 2XX
  • Redirection 3XX
  • Client Error 4XX
